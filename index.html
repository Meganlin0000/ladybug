01 我們怎麼工作

三步驟，把行銷裡的 bug 修好

好的行銷不是靠運氣，而是一套能重複的除錯流程。我們用做工程的邏輯做行銷。

// scan

找出問題

市場調查、競品與 SWOT 分析、消費者洞察。先搞懂品牌現在卡在哪、機會點在哪，而不是憑感覺猜。

// fix

解決問題

客製整合行銷企劃：數位廣告、公關、社群短影音與視覺設計一起規劃，把策略變成可執行的方案與預算。

// ship

放大亮點

以 O2O 串聯線上線下，落地成實體活動與展演，做成效追蹤與檢討，讓品牌以它獨有的方式被記住。
